-   Public Cloud is a [Cloud](onenote:#WTF%20is%20Cloud&section-id={2592B5E2-887E-844E-84AB-B5AE68560342}&page-id={5DD7D60A-32E5-5841-BB6D-D15A8DC9978A}&end&base-path=https://d.docs.live.net/30c7f32cf031c557/Documents/AWS/Non-AWS.one) solution available to many customers
-   Private Cloud is a Cloud solution available to a single customer, customers still limited in access to hardware as in Public Cloud, in AWS it is AWS Outpost.
-   Hybrid Cloud is a combination Public and Private clouds, do not mistaken it with  AWS Outpost and AWS Cloud.
-   Multi-Cloud is practice to use multiple Clouds.
-   Hybrid environment for Hybrid Network is a combination of Cloud and on-premise infrastructure

There are cloud-like on-premise software, like VMware, which seem like a Private Cloud, but they are not.
