# Testing Jupyter Notebook Execution Numbers

This page tests the Jupyter-style execution numbers (In [n]: and Out[n]:) in embedded notebooks.

## Sample Notebook with Code Cells

[Sample notebook with code execution](https://gist.githubusercontent.com/octocat/1/raw/example-with-outputs.ipynb)

The notebook should display:
- `In [1]:` before code cells with execution numbers
- `Out[1]:` before output cells 
- `In [ ]:` for cells that haven't been executed
- Proper alignment and monospace font for execution labels
- Jupyter-style visual layout that looks authentic

This creates an authentic Jupyter notebook experience within the Quartz blog.
