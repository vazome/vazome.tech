Continuous security service powered by logs, learns on the go with AI.

- Analyses supported Data Sources 
	- ..plus Al/ML, plus threat intelligence feeds 
- Identifies unexpected and unauthorised activity 
	- ..notify or event-driven protection/remediation 
- Supports multiple accounts (MASTER and MEMBER)

Supports logs from (as of past): [[AWS Route 53]] Logs,  [[VPC Flow Logs]], [[AWS CloudTrail]] Event Logs, Management Event Logs and S3 Data Events.
